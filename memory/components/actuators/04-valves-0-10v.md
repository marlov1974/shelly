# 0-10 V Valves

## Role

0-10 V valves are used for heating and cooling batteries or hydronic control.

## Control principle

Valve percentage is a command position, not guaranteed thermal power. Delivered effect depends on:

- water/brine temperature
- flow
- pressure differential
- coil/exchanger capacity
- air or floor-side conditions

## Safety

Cooling valves require condensation/dew point safeguards. Heating valves should coordinate with VVX and fan run status to avoid unwanted compensation or overshoot.
